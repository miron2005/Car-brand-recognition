# Car-brand-recognition
Этот проект посвящён распознаванию марок автомобилей с использованием сверточной нейронной сети (CNN). Модель обучена на датасете, содержащем изображения трёх марок автомобилей: Ferrari, Mercedes и Renault. Проект включает в себя обучение модели, сохранение её весов, а также подготовку скрипта для предсказания и архива для загрузки на демо-панель.
